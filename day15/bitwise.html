bitwise operator

it is used to perform bit wise operation

types :

  • -and
  • -or
  • -xor
  • -not
  • -left shift
  • -right shift
realtime use cases:
  • low -level data manipulation
  • data encoding
  • Encrypt algorithms
  • Data compression and encryption algorithms
  • Network protocols (e.g., IP address masking)
  • Graphics programming (e.g., color manipulation)
  • Embedded systems and hardware programming
